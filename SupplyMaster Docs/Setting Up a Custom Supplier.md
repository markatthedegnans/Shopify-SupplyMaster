If your supplier is not in SupplyMaster's list of built-in integrations, you can submit a **Custom Supplier** request. Our support team will assess the supplier and configure the integration on your behalf.

---

## When to Use a Custom Supplier

Use the Custom Supplier option when:

- Your supplier is not listed in the supplier type dropdown.
- Your supplier provides product data via a file format (CSV, Excel, XML) rather than a standard API.
- You have a private or niche supplier that requires a custom integration.

---

## How to Create a Custom Supplier

1. From the **Manage Suppliers** page, click **Add Supplier**.
2. Below the supplier type dropdown, click the **Create Custom Supplier** button.
3. Enter a **Supplier Name** (e.g., "My Supplier - Custom").
4. Fill in any available credentials. All three fields are optional:
	- **Username** — if the supplier provides one
	- **Password** — if the supplier provides one
	- **Additional Info** — any extra details, URLs, API keys, or notes that may help our team set up the connection
5. Click **Save Supplier**.

There is no **Test Connection** step for custom suppliers — the connection is configured by the support team after you save.

---

## What Happens Next

After you save a custom supplier, our support team is notified and will:

1. Assess the supplier and the data format available.
2. Configure the integration on the backend.
3. Email you an update within **3 business days**.

Once the setup is complete, products from your custom supplier will appear in your SupplyMaster account.

---

## Limitations

Custom suppliers work differently from built-in integrations. The following features are **not available** for custom suppliers:

| **Feature** | **Available?** |
| --- | --- |
| Auto Import / Automatic Sync | No — syncs are managed by the support team or via manual file upload |
| Product Fetch (pull from supplier API) | No — products are imported via file |
| Browse Catalog | No |
| Test Connection | No |
| Duplicate Supplier | No |
| Manage Orders | No |
| AI+ Enriched Data | No |
| Inventory Settings, Product Settings, Advanced Settings tabs | No — only the Connection Settings tab is shown |

Despite these limitations, custom suppliers still benefit from SupplyMaster's product management, field mapping (once configured by support), and Shopify integration.

## Need Help?

If you have questions about custom supplier setup or want to check on the status of your request, contact support on chat in the bottom-right corner of the app.
